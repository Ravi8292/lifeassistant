# Life Assistant – AI-Based Productivity Application

## 📌 Overview
Life Assistant is an AI-powered Spring Boot application that helps users manage daily tasks, reminders, and notes through REST APIs.

## 🛠 Tech Stack
- Java
- Spring Boot
- REST APIs
- Maven
- MySQL
- Hibernate / JPA
- Ollama (LLM integration)

## 🚀 Features
- Create, update, and delete tasks
- AI-assisted task suggestions
- RESTful API design
- Centralized exception handling
- Secure configuration (no secrets in repo)

## 🧩 Architecture
- Controller layer for REST APIs
- Service layer for business logic
- Repository layer using JPA/Hibernate

## ▶ How to Run
1. Clone the repository  
2. Configure database & env variables  
3. Run `LifeAssistantApplication.java`
